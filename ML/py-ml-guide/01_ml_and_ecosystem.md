## 머신러닝의 개념
- 애플리케이션을 수정하지 않고도 데이터를 기반으로 패턴을 학습하고 결과를 추론하는 알고리즘 기법

### 유형 (by 마스터 알고리즘)
- 기호주의: 결정 트리, etc..
- 연결주의: 신경망/딥러닝
- 유전 알고리즘
- 베이지안 통계
- 유추주의: KNN, 서포트 벡터 머신

### 단점
- 데이터에 너무 의존적 (garbage in, garbage out)
- 과적합 쉬움
- 복잡한 알고리즘으로 인해 도출된 결과에 대한 논리적인 이해가 어려움 (머신러닝 == 블랙박스)
- 데이터만 집어 넣으면 자동으로 최적화된 결과를 도출할 것 -> **환상!**
